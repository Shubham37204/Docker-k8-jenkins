## 📚 Learning Objectives

Through this repository, the following concepts are being explored:

* Understanding **Docker** and containerization
* Learning how **Jenkins** automates build and deployment processes
* Exploring **Kubernetes** for container orchestration
* Understanding the basic **CI/CD workflow**
* Practicing DevOps tools integration

## 🛠 Technologies Used

* **Docker** – Used to containerize applications
* **Kubernetes** – Used for managing and deploying containers
* **Jenkins** – Used for building CI/CD pipelines
* **GitHub** – Used for version control and code hosting
